# Codex Reconnecting Solve (Codex 重连修复工具)

## 📌 项目简介
在使用 Codex 的过程中，许多用户会遇到一个烦人的问题：软件启动或使用初期，总是会连续显示 5 次 "reconnecting..."（重新连接）之后才能正常工作。

本项目提供了一个简单有效的解决方案。仓库中的 `CODEX.exe` 程序专门针对此 bug 进行修复，帮你跳过漫长的重连等待，实现丝滑启动。

## 🚀 功能特点
* **一键修复**：解决 Codex 必须重连 5 次才能使用的问题。
* **简单易用**：无需复杂的配置，下载即用。

## 🛠️ 使用方法
1. 在本页面的文件列表中，点击 `CODEX.exe`。
2. 点击页面右侧的 **Download raw file** 按钮（通常是一个向下的小箭头图标）将程序下载到本地。
3. 双击运行 `CODEX.exe` 即可。
4. 需要本地先下载好Codex官方程序，二者不冲突。
